# 概述

---

- WooKong 发行的每一张 NFT 均获得全球传奇漫画家吴先生的官方授权
- WooKong Telegram 小程序集成了排行榜、邀请积分、积分兑换、NFT 铸造、NFT 上架等功能。
- 用户可以在 WooKong Telegram 小程序内完成所有活动和操作
- 运营后台管理，包含了集合、NFT、用户、订单等管理等；
- 产品使用的技术栈：Next.js、Monorepo、Node、Postgresql、Ton 链 API
